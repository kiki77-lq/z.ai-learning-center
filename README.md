# Zhipu AI Learning Center

一个现代化的 AI 学习平台，采用极简黑白配色方案，注重清晰的内容层级和现代科技感。

## 设计特点

### 视觉风格
- **纯黑白配色**：黑色背景配合白色和灰度文字，无任何彩色装饰
- **极简主义**：通过对比、留白和排版营造视觉层次，不依赖颜色区分
- **高对比度**：确保文字清晰可读，符合现代 Web 无障碍标准
- **专业高端**：适合专业 AI 学习平台的冷静、可信赖形象
- **大图文配对**：首页采用大尺寸图片与文字的组合布局

### 信息架构

#### 主要页面（完整内容页）
- **首页 (Home)**: 展示平台价值主张、学习阶段、核心能力和认证路径
- **课程 (Courses)**: 结构化学习路径，包含课程筛选和详细信息
- **认证 (Certification)**: 清晰的认证等级和进阶路径

#### 外部链接导航
- **模型 (Models)**: 链接到智谱AI开放平台 (https://open.bigmodel.cn/)
- **文档 (Documentation)**: 链接到开发者API文档 (https://open.bigmodel.cn/dev/api)

### 核心功能

#### 学习阶段体系
1. **初学者**: 基础 AI 和编程概念的引导式学习
2. **进阶者**: 通过实践练习提升工程技能
3. **实践者**: 真实项目应用和认证获取

#### 平台能力
- 结构化课程体系
- 技能认证路径
- 技术文档支持
- 开源模型探索
- 学习进度追踪

## 设计系统

### 色彩方案（纯黑白）
```css
--background: 0 0% 5%;         /* 纯黑背景 */
--foreground: 0 0% 98%;        /* 纯白文字 */
--primary: 0 0% 95%;           /* 白色按钮/强调元素 */
--card: 0 0% 10%;              /* 卡片背景（浅黑） */
--muted-foreground: 0 0% 60%;  /* 次要文字（中灰） */
--border: 0 0% 18%;            /* 边框（深灰）*/
```

**交互状态**
- 悬停：通过灰度变化（更亮或更暗）
- 激活：无色彩变化，仅透明度调整
- 焦点：浅灰色描边

### 排版
- **字体族**: Inter, system-ui, sans-serif
- **标题**: 5xl ~ 8xl (大标题), 3xl ~ 5xl (小节标题)
- **正文**: base ~ xl
- **行高**: relaxed (1.625) 确保可读性

### 间距
- **区块间距**: py-32 (128px) 主要区块
- **容器内边距**: px-4 (16px) 移动端, px-6 ~ px-8 桌面端
- **元素间距**: gap-4 ~ gap-8 根据内容密度调整

## 动画设计

### 页面过渡
- **淡入上升**: 元素从下方 30px 淡入
- **交错动画**: 子元素依次出现（0.15s 延迟）
- **视口触发**: 滚动到可视区域时触发动画

### 交互反馈
- **悬停效果**: scale(1.02), 颜色渐变
- **点击反馈**: 按钮压缩效果
- **平滑过渡**: 300ms ease-out

## 响应式设计

### 断点
- **移动端**: < 768px (单列布局)
- **平板**: 768px ~ 1024px (双列布局)
- **桌面**: > 1024px (三列/多列布局)

### 适配策略
- 移动端导航折叠为汉堡菜单
- 卡片网格自适应列数
- 文字大小和间距按屏幕尺寸缩放

## 维护说明

### 添加新课程
1. 更新 `src/pages/Courses/index.tsx` 中的 `courses` 数组
2. 添加课程封面图片
3. 确保难度等级和学习时长准确

### 修改主题色
1. 编辑 `src/index.css` 中的 CSS 变量（当前为纯黑白配色）
2. 所有颜色使用 HSL 格式，色相和饱和度为 0（灰度）
3. 调整明度（Lightness）来控制灰度深浅
4. 避免使用任何彩色值以保持极简美学

### 更新导航
1. 修改 `src/components/Layout.tsx` 中的 `navLinks` 数组
2. 内部页面使用 `path` + `Link` 组件
3. 外部链接使用 `path` + `external: true` + `<a>` 标签
